# Cel Projektu  

Stworzenie własnej internetnowej wizytówki.

## Struktura Serwisu

 index.html - plik/strona **główna** z której możemy uzyskać dostęp do pozostałych podstron takich jak:
* omnie.html,
* wyksztalcenie.html,
* zainteresowania.html,
* kontakt.html.

style.css - używany do stylowania html.
